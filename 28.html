Basic Information about the element

Name: Nickel
Symbol: Ni
Atomic Number: 28

1s2 2s2 2p6 3s2 3p6 3d10

The chemical element Nickel is classed as a Transition metal.
It was discovered in 1751 by Alex Cronstedt.
Number of Protons/Electrons: 28
Number of Neutrons: 31
Atomic Mass: 58.6934 amu
Melting Point: 1453.0 °C (1726.15 K, 2647.4 °F)
Boiling Point: 2732.0 °C (3005.15 K, 4949.6 °F)
Classification: Transition metal
Uses: Used for electroplating metal alloys, nickel-cadmium batteries
