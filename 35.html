Basic Information about the element

Name: Bromine
Symbol: Br
Atomic Number: 35

The chemical element bromine is classed as a halogen.
It was discovered in 1826 by Antoine J. Balard.
Number of Protons/Electrons: 35
Number of Neutrons: 45
Atomic Mass: 79.904 amu
Melting Point: -7.2 °C
Boiling Point: 58.78 °C
Classification: Halogen
Uses: Bromine is a poisonous element.
