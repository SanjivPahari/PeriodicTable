Basic Information about the element

Name: Lithium
Symbol: Li
Atomic Number: 3

1s2 2s1

The chemical element lithium is classed as an alkali metal.
It was discovered in 1817 by Johan Arfvedson.

Number of Protons/Electrons: 3
Number of Neutrons: 4
Atomic Mass: 6.941 amu
Melting Point: 180.54 °C
Boiling Point: 1347 °C
Classification: Alkali Metal
Uses: Batteries, Ceramics, Lubricants
